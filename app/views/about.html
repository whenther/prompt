A teleprompter for the 21st Century

This is an alpha version of a web app built for one purpose: scrolling text files at a steady rate. It's handy for musicians or speakers who have text that needs to zip by in a certain amount of time. It uses your browser's local storage, so your data will persist between restarts.

Add a text

First, click NEW on the nav bar to add a text file. There's a little required information.
name A memorable name, so you can find this later!
time The time (in milliseconds) to scroll your text
body The actual text. This can include basic HTML formatting tags (like p, hr, em, or strong).

Play a text

Choose the text name from the Prompts downdown menu in the nav bar.
Play Start or continue scrolling the text
Pause Pause scrolling. You can continue where you left off by pressing play again.
Top Scroll the text to the top, ready to be played again

Edit/Delete a text

Choose the text name from the Prompts downdown menu in the nav bar and press Edit.
Delete Permenantly delete this text.
Edit the text Feel free to edit any text.
